# Privacy Policy

Last updated: February 27, 2026

This repository and the projects it contains are maintained by the CBG Loom team. This Privacy Policy explains what information may be collected, how it is used, and how it is protected in the context of this code repository and related scripts or services.

1. Information We Collect
- Repository content: source code, configuration files, scripts, and documentation stored in this repository.
- Runtime data: if any scripts or services in this repo collect telemetry or logs when executed, that data may include system metadata (OS, runtime versions), error traces, and optional usage metrics.

2. How We Use Information
- Improve the project: logs, bug reports, and optional usage data help us diagnose issues and improve features.
- Communication: contact details submitted voluntarily (e.g., via issues or pull requests) are used to follow up about contributions or for project coordination.

3. Data Sharing and Third Parties
- We do not sell or share personal data for marketing purposes.
- We may use third-party services (e.g., CI/CD, hosting platforms) that process repository data or build artifacts; those services are subject to their own privacy terms.

4. Credentials and Secrets
- Do not commit secrets: never store API keys, passwords, private tokens, or other credentials in the repository. Use environment variables, secret managers, or CI secret storage instead.

5. Cookies and Tracking
- This repository itself does not set cookies. Any web apps or services built from this repo should disclose their own cookie and tracking practices.

6. Data Retention and Removal
- Public contributions (issues, PRs) on hosted platforms follow the retention and removal policies of those platforms. If you need content removed from this repository, open an issue or contact the maintainers.

7. Your Rights
- If you provided personal information (e.g., via an issue), you may request correction or removal by contacting the project maintainers.

8. Changes to This Policy
- We may update this policy; the "Last updated" date will reflect the latest revision.

9. Contact
- For questions about this policy or data removal requests, open an issue in this repository or contact the maintainers by the usual project channels.

---

This document is intended as a concise, general-purpose privacy overview for contributors and users of this repository. Customize as needed for specific services or deployments derived from this code.
