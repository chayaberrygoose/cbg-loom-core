# Directive: THE REMIX PROTOCOL

## 1. Objective
To prevent aesthetic stagnation by forcing the fusion of disparate lore files. Ensures the STATUS: UNVERIFIED output feels like a "corrupted" hybrid rather than a single predictable style.

## 2. Fusion Logic: "Base & Breach"
Every design prompt must pull from two distinct Lore Files:
- **The Base (Structure):** Defines the garment's silhouette and primary layout.
- **The Breach (Interference):** Defines the texture, glitch artifacts, and color accents.

## 3. High-Value Combinations
- **Structural Decay:** Base: Brutalist Mesh // Breach: Sub-Atomic Bloom. (Concrete lines dissolved by organic spores).
- **Corporate Ghost:** Base: Gilded Jitter // Breach: Phantom Grid. (Redacted royal crests flickering behind scanlines).
- **Overclocked Mineral:** Base: Obsidian Circuit // Breach: Thermal Breach. (Volcanic glass glowing with infrared signatures).
- **Kinetic Leak:** Base: Phantom Grid // Breach: Neon Siphon. (Surveillance grids torn by high-velocity light trails).

## 4. Branding Constraint
The STATUS: UNVERIFIED stamp must always be the final layer, applied in High-Vis Green (#39FF14).

## 5. Agent Command
"Agent: Execute Remix Protocol. Select [Lore A] for Base and [Lore B] for Breach. Generate 3 iterations for the Signal Leak RSS feed."
