# GRAPHICS

These are the types of images based on their function in the clothing design. Their specific content can vary. It is possible to create multiple types from the same source content. E.g. The idea of 'data glitch' could be expressed via any one of these types. 

## standalone

These are standalone images.

## textures

These are textures that can be used for trim or overlaid on top of other graphics

## tiles

These are graphics that are meant to be tiled.
