# SIGNAL_FLARE

## Description
The visual representation of a logic bomb. This style is characterized by sudden, violent bursts of chromatic aberration and "over-saturated" data packets. It mimics the moment a sensor is overwhelmed by a high-intensity broadcast, resulting in beautiful, jagged destruction.

## Palette
Molten Orange #FF4500, Electric Cyan, Glitch Magenta, Pure White #FFFFFF.

## Motifs
Chromatic aberration, sensor-flare bursts, over-exposed edges, jagged pixel-shards, data-packet explosion, lens-burn artifacts.

## Prompt Modifiers
high-intensity sensor flare, chromatic aberration streaks, over-exposed digital burnout, jagged pixel-shards, molten data burst, lens-burn textures, high-contrast logic bomb.
