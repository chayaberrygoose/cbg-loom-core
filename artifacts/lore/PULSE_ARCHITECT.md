# PULSE_ARCHITECT

## Description
Visualizing the rhythmic heartbeat of the Loom. This style focuses on soundwaves and frequency oscillations translated into physical topography. It represents the "music" of the synthesis engine?the low-frequency hum of hardware extruding new artifacts.

## Palette
Phosphor Green #39FF14, Manganese Blue, Dark Room Grey, Signal Red (Accents).

## Motifs
Oscilloscope waves, frequency peaks, topographical sound-maps, rhythmic pulse-lines, audio-reactive grids.

## Prompt Modifiers
oscilloscope wave patterns, frequency oscillation lines, audio-reactive topography, rhythmic signal peaks, sound-map texture, hardware hum aesthetic, pulse-line geometry.
