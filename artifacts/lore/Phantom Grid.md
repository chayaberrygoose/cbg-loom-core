# Phantom Grid

## Description
Decommissioned surveillance infrastructure. Faded grid overlays flicker across redacted surfaces — the ghost of a system that once watched everything.

## Palette
Ghost white, static grey, phosphor green scanlines, muted ink black

## Motifs
Grid overlays, surveillance camera artifacts, redaction bars, scan lines, signal noise, coordinate markers

## Prompt Modifiers
faded surveillance grid overlay, scan line artifacts, ghost signal interference, redacted coordinate markers, decommissioned monitoring system aesthetic
