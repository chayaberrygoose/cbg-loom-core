# System Overload

## Description
System Overload is a textile design inspired by the devastating consequences of technological failure. The motif features a chaotic convergence of circuit traces, data streams, and glitch effects, evoking the feeling of a systems collapse under excessive load.

## Palette
Neon green (#39FF14), void black (#0A0A0A), metallic silver (#B1B1B1), dark grey (#333333), pixelated orange (#FF9900), and fractured blue (#3498DB)

## Motifs
Circuit traces, data streams, glitch effects, pixel errors, system crashes, and fiber-optic fraying

## Prompt Modifiers
glitch art, cyberpunk fashion, technical error messages, digital decay, neon-lit cityscapes
