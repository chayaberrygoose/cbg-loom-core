# Ghost Wire

## Description
The Ghost Wire is an anomalous entity that has emerged from a corrupted sector of the Loom's source code. It manifests as a digital ghost, rendered in phosphor green wireframe, which should not exist within the stable system.

## Palette
- #33CCFF (phosphor green)
- #1A1D23 (dark industrial grey)
- #212121 (dark cyan)
- #353839 (cool blue-black)
- #434A54 (metallic silver)

## Motifs
digital wireframe, glitch artifact, sensor arrays, data transmission patterns

## Prompt Modifiers
phosphor green glow, pixelated textures, wireframe geometry, data transmission lines, digital decay
