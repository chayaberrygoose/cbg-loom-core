# System Lockdown

## Description
The subject is trapped in a geofence loop, requiring 50 miles of kinetic transit to maintain system access while being physically present in a distant geographical sector. This creates a paradoxical state where the individual exists as both local and remote, with their creative output only manifesting upon return to the "Home Sanctum" or entering an elliptical flow state.

## Palette
- Asphalt Black (#1C1C1C)
- Cubicle Fluorescent (#E5E4E2)
- Home Sanctum Amber (#FFBF00)

## Motifs
- Geofence Loop visual representation
- Badge reader connected to a long, winding cable
- Circuit board patterns with circuit breaks
- Data stream anomalies
- Glitch effects on screen borders

## Prompt Modifiers
(circuit boards, data transmission, system failure, grid lock)
