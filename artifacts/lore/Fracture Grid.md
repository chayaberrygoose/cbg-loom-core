# Fracture Grid

## Description
Fracture Grid is a textile designed to harness the dissonance between industrial systems and human perception. By integrating circuit patterns and data streams, this fabric embodies the tension between technological advancement and existential unease. The result is a wear that seems to vibrate with the hum of machinery.

## Palette
Neon green (#39FF14), void black (#0A0A0A), metallic silver (#B1B1B1), dark grey (#333333), circuit red (#FF3737)

## Motifs
Circuit traces, data streams, glitch effects, binary code overlays, industrial piping patterns

## Prompt Modifiers
glitch art, pixelated textures, worn machinery aesthetics, low-res digital prints
