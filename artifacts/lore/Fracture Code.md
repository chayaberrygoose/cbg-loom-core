# Fracture Code

## Description
Fracture Code is a textile brand engineered for the modern rebel. By integrating advanced materials and precision-cutting techniques, we create garments that not only defy convention but also serve as a beacon of resistance against the monotony of commercialized conformity.

- Ink-stained black (#121212)
- Circuitry grey (#454545)
- Binary blue (#2F4F7F)
- Datastream white (#FFFFFF)
- System failure red (#E74C3C)
