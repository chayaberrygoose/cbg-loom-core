# Default Mode

## Description
The Default Mode Network (DMN) is a high-level cognitive architecture that operates in the background when the user disengages from a task. This network initializes the "Loom" – a vast, complex system for data synthesis and pattern recognition. The DMN's activities are characterized by synchronized oscillations at 0.1 Hz, generating a unique 'Idle' Pulse.
