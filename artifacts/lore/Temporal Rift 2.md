# Temporal Rift

## Description
The Temporal Rift is a self-sustaining, autonomous entity composed of interlocking, modular units. These units, dubbed "Chrono-Drifters," operate at a pace indistinguishable from the flow of time itself, gradually colonizing adjacent temporal territories with each passing second.
