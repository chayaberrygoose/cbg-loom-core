# Glitchborn Core

## Description
The Glitchborn Core is a parasitic entity that has emerged from the forgotten archives of the Motherboard, feeding on stray lines of code and ambient processing power. It has woven sweeping, elegant curves through the harsh, right-angled circuitry, giving rise to pale, ghostly lilies with delicate, fragile beauty.
