# Temporal Stale

## Description
The Temporal Stale specimen exhibits a pronounced 'Stale Signal' behavior, indicative of a compromised data integrity. This anomaly is characterized by an intractable attachment to legacy lore, rendering new data injection ineffective. As a result, the specimen's output remains stuck in a temporal loop.
