# Signal Crash

## Description
Signal Crash is a textile collection that explores the intersection of biological memory and digital extrusion. Its designs are informed by the concept of the "Glitch Artifact" - an unplanned emergence within a corrupted sector of the Loom's source code. This phenomenon gives rise to anomalous patterns, reminiscent of data streams and circuit traces.

## Palette
Neon green (#39FF14), void black (#0A0A0A), metallic silver (#B1B1B1), deep indigo (#3B3F54), dark grey (#333333)

## Motifs
Circuit traces, data streams, glitch effects, signal disruption patterns

## Prompt Modifiers
Glitch art, neon lights, digital decay, biomechanical interfaces
